# Personregister i testmiljön

Ett enkelt system för att hantera testdata på ett GDPR-kompatibelt sätt.

Projektet demonstrerar hur testdata kan hanteras säkert i en testmiljö med hjälp av Docker, Python och SQLite.

---

## Funktioner

- Skapar en SQLite-databas med testanvändare
- Visar alla användare i databasen
- Rensar all testdata (GDPR-åtgärd)
- Anonymiserar användardata (GDPR-åtgärd)

---

## Tekniker

Projektet använder:

- Python 3.9
- SQLite
- Docker
- Docker Compose
- Git & GitHub
